# Quest4You 🎯

Plataforma de questionários de autoconhecimento sobre sexualidade, fetiches e compatibilidade.

## ✨ 100% Grátis

Todos os questionários e funcionalidades são **completamente gratuitos** nesta fase inicial!

## 📝 Descrição

Quest4You é uma aplicação web que permite aos utilizadores:
- Responder a questionários individuais sobre diversos temas
- Obter resultados imediatos e personalizados
- Descobrir compatibilidade com outros utilizadores (Smart Match)

## 📋 Questionários Disponíveis (50 perguntas cada)

1. **💕 Vanilla ou Kink** - Espectro de intensidade sexual
2. **🌈 Orientação Sexual** - Escala de Kinsey melhorada
3. **👀 Stag/Cuckold** - Dinâmicas de voyeurismo
4. **💜 Swing/Poliamor** - Abertura relacional
5. **🔥 Fetiches & Kinks** - Descoberta de interesses
